# MIPS-Assembly-Projects
This reposiroty showcases the lab works done during cs 224 course with MIPS Assembly.
repository structure is as follows
  1. Basics & arithmetic: Fundamental programs focusing on system calls and core operations.
         helloWorld: basic output using system calls
         mathCalculation: simple integer arithmetic and user input
         arithmeticOperations: Complex expression evaluation using subroutines.
  2. Arrays &logic: programs showing memory management and iterative logic
         swapArray: in place array element swapping.
         simpleMenu: Menu interface for array analysis (max, frequency and divisibility)
         frequency_array:dynamic memory allocation and frequency counting logic
  3. Bit manipulation: low level logic focusing on binary data processing
          lab2-hamming: calculating hamming distance between two values
          lab2-reverse: bit by bit reversal of integer contents
   4. data structures & recursion: advanced memory management and recursive algorithms.
          recursiveDivision: int division implemented by recursive subtraction
          displayReverse: recursive traversal and printing of linked lists
          summaryLinkedList: dynamic creation and data aggregation of linked list nodes
  5. advanced systems logic: complex architectural implementations and instruction parsing
           lab6.asm: matrix memory mapping using row major and columnmajor summation.
            registerCount: A specialized subprogram that parses binary instructions to count specific register usage.
